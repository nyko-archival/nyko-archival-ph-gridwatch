# Screenshots

Add three PNG screenshots of the running dashboard here, then uncomment the
image block in the project `README.md` (under "Preview").

Expected filenames:

| File | Page to capture |
|------|-----------------|
| `executive.png`       | Executive dashboard (All Grids) |
| `energy-security.png` | Energy Security (All Grids, the score cards) |
| `generation.png`      | Generation (fuel mix) |

How to capture: run the app (`streamlit run app.py`), open each page, and take a
screenshot of the browser viewport. Keep them reasonably sized (under ~500 KB each).
